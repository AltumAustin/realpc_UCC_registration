### 2. AUTHORIZATION MODEL
**Model:** Public Portal with Optional Account Registration  
**Confidence Level:** ✓ High

Idaho operates an **open access public portal** (sosbiz.idaho.gov) with the following characteristics:

- **Free searches available** without account registration
- **Account required for filing** - As of February 25, 2019, users must create a SOSBiz account to file liens online
- **No formal approval program** - Account creation is self-service through the portal
- **Optional subscription services** available for bulk data access

### 5. FEE SCHEDULE

#### Search Fees
- **Non-certified search (online):** $3.00
- **Exempt users (government):** $0.00 (must be logged in)
- **Certified search:** $6.00 (one notice type) or $10.00 (multiple notice types)

#### Filing Fees
- **Online UCC-1/UCC-3:** $3.00 filing + $1.00 access = $4.00 total
- **Paper UCC-1/UCC-3 (1-2 pages):** $6.00 + $20.00 manual = $26.00 total
- **Paper UCC-1/UCC-3 (3+ pages):** $12.00 + $20.00 manual = $32.00 total

#### Subscription Services (Bulk Data)
- **Master List Registration:** $30.00
- **Annual Subscription:** $120.00
- **Per Distribution:** $8.00
- **UCC Data/Images Extract:** $125.00 per extract

#### Duration
- **UCC-1 Filing Duration:** 5 years from date of filing

### SPECIAL NOTES

1. **Government Fee Exemptions:** Government entities can obtain fee-exempt status by:
   - Using Access Idaho subscription account (automatic waiver), OR
   - County governments: Email username and county name to [email protected]

2. **Online vs. Paper:** Online filing strongly encouraged - saves $20 manual processing fee and receives faster processing

3. **Rejection Reasons:** Filings can be rejected for:
   - Missing required information
   - Insufficient filing fee
   - Fraudulent appearance (per Idaho Code 28-9-516A)

4. **System Update:** Major system change occurred February 25, 2019, requiring new SOSBiz account creation
